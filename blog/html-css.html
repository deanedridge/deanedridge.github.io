What is the difference between margin, border, and padding?

When one is new to client-side webdevelopment one will no doubtedly be wanting to learn the basics of layout and formating of web pages. One of the first things one will encounter will be what is the difference between margin, border, and padding?

In order to do this, one must learn something called the box-model, or the CSS box-model. Below is a diagram of the CSS box-model.


Image courtesy of Matthias Apsel - Own work, CC0, https://commons.wikimedia.org/w/index.php?curid=34932417

So, if we go from the inside to the outside: first, the padding goes inside the content box; like an old fashioned wooden chest might have padding inside it to pretect its contents from being damaged. Secondly we have the border which is placed on the outside of the content box. Thirdly, on the outside of the border, the margin is placed. That is the difference between margin, border, and padding.
